回学习台 第一课

核心提纲

  1. ATAM 关注多质量属性权衡，输出风险点、非风险点、敏感点、权衡点。
  2. SAAM 更强调场景驱动的可修改性分析。
  3. 效用树是质量属性优先级排序工具。
  4. 案例答题要落到“为什么这是风险/权衡”，不要只写名词。
