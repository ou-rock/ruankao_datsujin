回学习台 第一课

核心提纲

  1. 性能：响应时间、吞吐量、负载、延迟、资源调度。
  2. 可用性：故障、宕机、备用、重定向、恢复时间、主动冗余。
  3. 安全性：授权、审计、攻击检测、报警、追踪。
  4. 可修改性：变更、扩展、替换、接口稳定、信息隐藏。
